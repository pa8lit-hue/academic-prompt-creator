# academic-prompt-creator
เครื่องมือช่วยวิเคราะห์งานวิจัยด้วย AI

แอปพลิเคชัน "AI Research Assistant | ว.ป๋วยฯ มธ." พัฒนาขึ้นเพื่อแก้ปัญหาความยุ่งยากในการใช้งาน AI ที่นักศึกษามักต้องเสียเวลาคิดคำสั่ง (Prompt) ที่ซับซ้อน และเผชิญปัญหา AI ตอบนอกลู่นอกทางหรือสร้างข้อมูลอ้างอิงปลอม (Hallucination)

แอปนี้จึงถูกออกแบบมาให้เป็น "พี่เลี้ยงอัจฉริยะ (AI Mentor)" ที่ฝังโครงสร้างคำสั่งขั้นสูงและกฎเหล็กทางวิชาการไว้เบื้องหลัง ผู้ใช้เพียงแค่กรอกข้อมูลสั้นๆ ระบบก็จะสร้างคำสั่งที่แม่นยำให้โดยอัตโนมัติ ช่วยจัดระเบียบการทำงานตามเส้นทางการทำวิจัยอย่างเป็นระบบ ตั้งแต่การประเมินหัวข้อไปจนถึงการจำลองสอบป้องกัน

🎯 ทำให้นักศึกษาไม่ต้องปวดหัวกับการเขียน Prompt เอง สามารถดึงศักยภาพของ AI มาช่วยงานวิชาการได้อย่างสะดวก รวดเร็ว และได้ผลลัพธ์ที่ลึกซึ้งน่าเชื่อถือตามมาตรฐานวิชาการอย่างแท้จริง
